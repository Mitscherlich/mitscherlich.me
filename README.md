## Quick Start

1. Download

via `git clone` from this repo:

```bash
$ git clone https://github.com/Mitscherlich/mitscherlich.me.git
```

2. Install dependences

via `npm` or `yarn`:

```bash
$ npm install
# or
$ yarn install
```

3. Run!

via scripts already defined in `package.json`:

```bash
$ npm run serve
# yarn will also be ok
$ yarn serve
```

4. Build vendor

via scripts already defined in `package.json`:

```bash
$ npm run build
# yarn will also be ok
$ yarn build
```
